# Contribuer à WSSP-Forensics

Merci de votre intérêt pour ce projet de recherche du **Lichen-Collectives**. Nous encourageons la collaboration scientifique rigoureuse pour mettre en lumière les mécanismes opaques des systèmes d'exploitation.

## Philosophie de Contribution

Ce projet n'est pas seulement technique, il est éthique. Nous visons la transparence et la sécurité des utilisateurs.
Toute contribution doit respecter les principes suivants :

1.  **Rigueur Scientifique** : Les affirmations doivent être étayées par des logs, des dumps mémoire ou des équations vérifiables.
2.  **Neutralité** : L'analyse est technique. Bien que les implications soient géopolitiques, le code et la documentation doivent rester factuels.
3.  **Open Source** : Tout outil développé ici doit rester libre et auditable.

## Comment Contribuer

* **Forkez** le projet.
* **Signalez des anomalies** : Si vous observez des incohérences mathématiques dans vos outils Windows, ouvrez une Issue avec le tag `evidence`.
* **Proposez du code** : Scripts de détection (Python/PowerShell), profils Volatility, ou analyses de CVE.

Nous acceptons les Pull Requests qui améliorent la précision des modèles mathématiques ou qui ajoutent des méthodes de détection de processus fantômes.

*Note : Ce dépôt est surveillé. Tout code malveillant ou tentative d'obfuscation sera rejeté.*
